# Auto Video Downloader

Auto Video Downloader is a tool that automates downloading videos from various platforms using **yt-dlp** and **FFmpeg**.

## Included Tools

### yt-dlp

This project includes the **yt-dlp.exe** tool, which is an open-source command-line program to download videos from YouTube and other video platforms.

- **Official Repository**: [yt-dlp GitHub Repository](https://github.com/yt-dlp/yt-dlp)
- **License**: GPL-2.0 License

yt-dlp allows you to download, extract, and process video/audio content. Please ensure you use it in accordance with the licenses and the platform's terms of service. For personal or educational use, it is generally safe to use, but avoid redistributing copyrighted content without permission.

---

### FFmpeg

This project also includes **ffmpeg-2025-09-28-git-0fdb5829e3-full_build**, a powerful open-source multimedia framework used for decoding, encoding, transcoding, and streaming audio and video files.

- **Official Repository**: [FFmpeg GitHub Repository](https://github.com/FFmpeg/FFmpeg)
- **License**: LGPL-2.1 or GPL-2.0 (depending on build configuration)

FFmpeg is used in this project to process video/audio streams efficiently. By including FFmpeg, Auto Video Downloader can handle format conversion, merging, or other media processing tasks.

---

## Usage

1. Place the `yt-dlp.exe` and `ffmpeg-2025-09-28-git-0fdb5829e3-full_build` files in the same directory as your Auto Video Downloader script.
2. Run your script to download and process videos.
